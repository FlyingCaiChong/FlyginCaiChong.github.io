    # pnpm 的使用

    动机:

    1. 节省磁盘空间并提升安装速度
    2. 创建非扁平的 node_modules 目录

    # 安装

    通过 npm 安装

    npm install -g pnpm
    1

    通过 homebrew 安装

    brew install pnpm
    1

    # 设置源

    // 查看源
pnpm config get registry
// 切换淘宝源
pnpm config set registry http://registry.npm.taobao.org
    1
    2
    3
    4

    # 添加依赖包

    保存到 dependencies 配置项下

    pnpm add sax
    1

    保存到 devDependencies 配置项下

    pnpm add -D sax
    1

    保存到 optionalDependencies 配置项下

    pnpm add -O sax
    1

    安装软件包到全局环境中

    pnpm add -g sax
    1

    安装标记为 next 的版本

    pnpm add sax@next
    1

    安装指定版本 3.0.0

    pnpm add sax@3.0.0
    1

    # 更新依赖包

    pnpm update
    1

    # 移除依赖包

    pnpm remove
    1

    # 安装项目的全部依赖

    pnpm install
    1
